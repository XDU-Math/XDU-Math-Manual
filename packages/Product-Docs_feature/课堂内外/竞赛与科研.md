# 竞赛与科研

## 综测

综测这么多年来不知困宥了多少学子……

首先，你需要端正对综测的态度。综测不应该成为目的，只能是你在主干道上顺路摘的果子。数统院国奖名额稀少，卷的届你花再多精力都不一定成功，摆的届可能不用做啥就国奖了。而你不摆的话（把 70% 以上的非睡眠时间放在学习上），基本上奖学金是没跑的。国奖虽然有 1w，但想想你以后一个月就能赚 1w，也没必要冒风险去特意追求。
